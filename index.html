1. Seletores Básicos Universal: * Seleciona todos os elementos da página. Ex: * { margin: 0; } Tipo/Tag: elemento Seleciona todos os elementos de um tipo específico. Ex: p { color: red; } Classe: .classe Seleciona todos os elementos com a classe especificada. Ex: .destaque { font-weight: bold; } ID: #id Seleciona o elemento com o ID específico. Ex: #menu { background: gray; }
